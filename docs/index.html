High-Performance Deformable Registration

A unified 2D and 3D medical image registration framework using deep-learning-based similarity metrics, accelerated by PyTorch and JAX.

Framework Architecture

⚡

Dual Backends

Optimize transformation grids using PyTorch autograd or JAX accelerated gradient descent. Toggle with a single backend parameter.

🧠

Perceptual Metrics

Replace classic metrics (LNCC, Mattes Mutual Information) with high-fidelity deep feature matching (VGG19, DINOv2, Swin UNETR).

🌉

Zero-Copy Bridge

Unify JAX registration and PyTorch neural networks via DLPack, allowing gradient passing directly in memory without CPU copies.

Core Similarity Metrics

Syntx incorporates several state-of-the-art models as modular similarity metrics, wrapping them into the FeatureSpaceLoss class:

VGG19

Standard 2D VGG19 features, or 3D orthogonal slicing mode to compute LNCC across multiple planes.

DINOv2

Pretrained Vision Transformers (ViT-S/14 or ViT-B/14) containing dense semantic features.

ResNet-10

Highly optimized, shallow 2D/3D ResNet for quick registrations and lower memory footprints.

Swin UNETR

3D self-supervised encoder pretrained on massive clinical datasets (MONAI zoo) with dynamic scale handling.

Quick Example

from syntx import registration
from syntx.features import SwinUNETRExtractor, FeatureSpaceLoss

# 1. Instantiate the extractor and loss
swin_ext = SwinUNETRExtractor(feature_layers=[4], img_size=(96, 96, 96))
similarity_metric = FeatureSpaceLoss(extractor=swin_ext, mode='lncc_3d')

# 2. Run deformable registration using JAX and DLPack
result = registration(
    fixed=fixed_image,
    moving=moving_image,
    type_of_transform='SyNTo',
    backend='jax',
    reg_iterations=[20, 10, 0],
    levels=[4, 2, 1],
    similarity_metric=similarity_metric
)
