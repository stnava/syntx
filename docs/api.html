API Reference

Feature Extractors

All extractors inherit from the FeatureExtractor base class and implement the extraction interface.

class VGG19Extractor(FeatureExtractor)

Extracts multi-scale features from 2D slices or orthogonal planes of 3D volumes using pretrained VGG19 weights.

Parameters:

  • feature_layers: List of layer indices to extract features from. Defaults to [8].

class DINOv2Extractor(FeatureExtractor)

Wraps Meta's pretrained self-supervised Vision Transformers (ViT-S/14 or ViT-B/14).

Parameters:

  • version: Extractor backbone size ('vits14' or 'vitb14').
  • feature_layers: List of transformer layer indices.

class ResNet10Extractor(FeatureExtractor)

A lightweight 2D/3D ResNet for high-speed convolutional feature extraction.

Parameters:

  • dim: Spatial dimension (2 or 3).
  • feature_layers: List of ResNet blocks to output.

class SwinUNETRExtractor(FeatureExtractor)

MONAI Swin UNETR self-supervised 3D encoder with weights cached locally. Handles input resizing automatically.

Parameters:

  • feature_layers: List of downsampling layers in [1, 2, 3, 4].
  • weights_path: Path to local PyTorch checkpoint. Auto-downloads from Project MONAI repository if absent.
  • img_size: Tuple specifying expected isotropic/non-isotropic input size (e.g. (96, 96, 96)).

Feature Space Loss

class FeatureSpaceLoss(nn.Module)

Evaluates a structural similarity metric (such as LNCC) on extracted multi-resolution feature maps instead of raw image intensities.

Parameters:

  • extractor: An instance of a FeatureExtractor class.
  • mode: Dimensional loss mode (e.g. 'lncc_3d' or 'lncc').
  • lncc_window: Neighborhood window size (defaults to 9).

DLPack Autograd Bridge (JAX-PyTorch)

Helper functions defined in syntx.syn_jax to support zero-copy tensor mapping and gradient propagation between PyTorch and JAX backends:

to_torch_tensor(x_jax)

Directly maps a JAX array to a PyTorch tensor via the DLPack protocol (zero CPU memory copy).

to_jax_array_dl(x_torch)

Directly maps a PyTorch tensor back to a JAX array via the DLPack protocol.

make_pytorch_loss_jax(pytorch_loss_fn)

Wraps any PyTorch loss function or nn.Module into a JAX-grad-compatible function with custom Vector-Jacobian Product (VJP) backpropagation.
